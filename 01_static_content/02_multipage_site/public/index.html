About Paths

We are using root relative paths. You can tell because they start with the / character. The site root for our Express app is public, so it will start the path there and try to find files based on the given path.

  • / = start at the root and serve the default file.
  • /about.html = start at root and serve about.
  • /contact.html start at root and serve contact.
  • /css/styles.css needs to include the css folder in the path, or else it would not work. Try removing it and see what happens.
